Sparka uses standard [shadcn theming](https://ui.shadcn.com/docs/theming) with CSS variables. If you're familiar with shadcn, everything works as expected.

## Quick start

1. Open `app/globals.css`
2. Edit the CSS variables in `:root` (light) and `.dark` (dark)
3. Key variables: `--background`, `--foreground`, `--primary`, `--secondary`, `--accent`, `--muted`

## Generate a theme

Use the [shadcn theme generator](https://ui.shadcn.com/themes) to create a palette, then copy the CSS variables into `globals.css`.

## Adding components

New shadcn components (`bunx shadcn@latest add <component>`) automatically use your theme variables. No extra config needed.
