## Overview

ChatJS uses Tailwind CSS 4 and Shadcn/UI for styling. Customize the look and feel by modifying CSS variables and component styles.

## Quick Start

### Colors

Edit `app/globals.css` to change the color scheme:

```css
:root {
  --primary: 239 84% 67%;
  --primary-foreground: 0 0% 100%;
  /* ... */
}
```

### Components

Shadcn/UI components in `components/ui/` can be customized directly.
