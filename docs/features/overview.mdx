## Advanced Patterns

These are documented in [Patterns](/patterns/index):

- **Resumable Streams**: Continue generation after disconnection via Redis
- **Branching**: Fork conversations, explore alternative paths
- **Syntax Highlighting**: Beautiful code blocks with theme support
- **Credit Tracking**: Monitor and limit AI usage per user
