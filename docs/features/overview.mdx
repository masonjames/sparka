ChatJS ships with production-ready features. Toggle them in `chat.config.ts`.

<CardGroup cols={2}>
  <Card title="Web Search" icon="globe" href="/features/web-search">
    Real-time search via Tavily or Exa. Ground responses in current information.
  </Card>
  <Card title="Deep Research" icon="magnifying-glass-chart" href="/features/deep-research">
    Multi-step research agent that investigates topics and produces comprehensive reports.
  </Card>
  <Card title="Code Execution" icon="terminal" href="/features/code-execution">
    Run Python in secure sandboxes. Execute code snippets inline.
  </Card>
  <Card title="Image Generation" icon="image" href="/features/image-generation">
    Generate and edit images with dedicated models or multimodal LLMs. Iterative
    editing without re-uploading.
  </Card>
  <Card title="MCP" icon="plug" href="/features/mcp">
    Model Context Protocol support. Extend capabilities with external tools.
  </Card>
  <Card title="Canvas" icon="pen-nib" href="/features/canvas">
    Create and edit text documents, code, and spreadsheets with AI assistance.
  </Card>
  <Card title="Attachments" icon="paperclip" href="/features/attachments">
    Drag and drop images, PDFs, and documents. Multimodal conversations out of
    the box.
  </Card>
  <Card title="Sharing" icon="share" href="/features/sharing">
    Share conversations with public links. Control visibility per chat.
  </Card>
  <Card title="Branching" icon="code-branch" href="/features/branching">
    Fork conversations and explore alternative paths with sibling navigation.
  </Card>
</CardGroup>
