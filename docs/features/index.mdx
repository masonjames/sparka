## AI Tools

| Tool | Description | Requires |
|------|-------------|----------|
| Web Search | Real-time search via Tavily/Exa | `TAVILY_API_KEY` |
| Code Execution | Python/JS sandboxes | `integrations.sandbox` |
| Image Generation | AI image creation | Model capability |
| Document Creation | Generate docs, spreadsheets | Built-in |
| Deep Research | Multi-step research with citations | Built-in |
