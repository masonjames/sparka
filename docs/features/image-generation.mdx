## Overview

Generate images directly in chat using AI models like DALL-E and Stable Diffusion.

## Quick Start

Image generation requires OpenAI integration in `lib/config.ts`:

```ts
integrations: {
  openai: true, // Requires OPENAI_API_KEY
}
```

## Environment Variables

```bash
OPENAI_API_KEY=your_openai_key
```
