## Overview

Execute Python and JavaScript code in secure sandboxed environments directly from chat.

## Quick Start

Code execution is enabled via sandbox in `lib/config.ts`:

```ts
integrations: {
  sandbox: true, // Vercel-native, no key needed
}
```
