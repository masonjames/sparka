## Overview

Enable your AI to search the web in real-time and provide answers with citations.

## Quick Start

Enable web search in `lib/config.ts`:

```ts
integrations: {
  webSearch: true, // Requires TAVILY_API_KEY or FIRECRAWL_API_KEY
}
```

## Environment Variables

```bash
TAVILY_API_KEY=your_tavily_key
# or
FIRECRAWL_API_KEY=your_firecrawl_key
```
