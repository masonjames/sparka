ChatJS ships with production-ready AI capabilities. Toggle them in `chat.config.ts`.

## Tools

<CardGroup cols={2}>
  <Card title="Web Search" icon="globe" href="/capabilities/web-search">
    Real-time search via Tavily or Exa. Ground responses in current information.
  </Card>
  <Card title="Code Execution" icon="terminal" href="/capabilities/code-execution">
    Run Python in secure sandboxes. Execute code snippets inline.
  </Card>
  <Card title="Image Generation" icon="image" href="/capabilities/image-generation">
    Generate and edit images with dedicated models or multimodal LLMs. Iterative
    editing without re-uploading.
  </Card>
  <Card title="MCP" icon="plug" href="/capabilities/mcp">
    Model Context Protocol support. Extend capabilities with external tools.
  </Card>
</CardGroup>
