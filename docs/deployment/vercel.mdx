## Overview

Deploy ChatJS to Vercel with automatic CI/CD, edge functions, and integrated storage.

## One-Click Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/franciscomoretti/chatjs)

## Manual Setup

1. Fork/clone the repository
2. Import to Vercel
3. Add environment variables
4. Deploy

## Required Integrations

- **Neon Postgres** or **Vercel Postgres** - Database
- **Vercel Blob** - File storage
- **AI Gateway** - AI provider access
